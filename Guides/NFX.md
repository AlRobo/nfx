# NFX
Server UNISTACK framework.

License: Apache 2.0

This framework is written in C# from scratch and runs on Windows and Linux/Mono servers.

**NUGET**:
 https://www.nuget.org/packages/NFX/
 
 `pm> install-package NFX`
